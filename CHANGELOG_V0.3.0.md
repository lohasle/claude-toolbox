### 新增功能

- ✅ **配置格式化工具** (`scripts/format-config.sh`)

  - 格式化 YAML 文件
  - Markdown 文档检查
  - 生成格式化报告

- ✅ **Prompt 模板库** (`prompts/prompt-templates.md`)
  - 快速开始模板
  - 功能开发模板
  - 代码审查模板
  - Bug 修复模板
  - 文档生成模板
